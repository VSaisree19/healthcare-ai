## 📌 About the Project

The **AI-Powered Smart Healthcare Disease Prediction System** is an intelligent web application that predicts diseases at an early stage using **Artificial Intelligence** and **Machine Learning** techniques.

The system analyzes patient health data such as age, blood pressure, glucose level, BMI, cholesterol, and other clinical parameters to predict the possibility of diseases like **Diabetes**, **Heart Disease**, **Hypertension**, and more — instantly and accurately.

## 🎯 Features

- ✅ Predicts 5 health conditions: **Diabetes, Heart Disease, Hypertension, At Risk, Healthy**
- ✅ Shows **confidence percentage** of prediction
- ✅ Gives **personalized health recommendations**
- ✅ Clean and simple **user-friendly UI**
- ✅ Powered by **Random Forest** machine learning algorithm
- ✅ **91% accuracy** on test data
- ✅ Instant results — no waiting

## 🧠 Machine Learning Details

| Property | Details |
|----------|---------|
| Algorithm | Random Forest Classifier |
| Training Samples | 1000 patient records |
| Features Used | 9 health parameters |
| Model Accuracy | ~91% |
| Libraries | Scikit-learn, Pandas, NumPy |

### Input Features

| Feature | Description |
|---------|-------------|
| Age | Patient age in years |
| Gender | Male / Female |
| Blood Pressure | Systolic BP in mmHg |
| Glucose Level | Blood glucose in mg/dL |
| Cholesterol | Cholesterol level in mg/dL |
| BMI | Body Mass Index |
| Heart Rate | Heart rate in bpm |
| Smoking | Smoking status (Yes/No) |
| Family History | Family disease history (Yes/No) |

### Output (Disease Prediction)

| Disease | Risk Level |
|---------|-----------|
| 🩸 Diabetes | High |
| ❤️ Heart Disease | High |
| 🔴 Hypertension | Medium |
| ⚠️ At Risk | Medium |
| ✅ Healthy | Low |

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/VSaisree19/healthcare-ai.git
cd healthcare-ai
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the app
```bash
streamlit run app.py
```

### 4. Open in browser
```
http://localhost:8501
```
